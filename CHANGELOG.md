# Changelog

## v0.6.0

- 新增全域搜尋（⌘K）與快速跳轉 Issue 功能，並記錄最近瀏覽
- Issue 詳情頁新增留言與引用功能
- Issue 詳情頁新增子議題、相關議題清單
- Issue 詳情頁新增追蹤與取消追蹤功能
- Dashboard 新增數字摘要統計卡片與「追蹤中」tab
- Issue 列表支援欄位排序、顯示更新時間，新增建立者與被分派者欄位
- 修復更新 Issue 後誤報認證失敗的問題
- 修復 macOS 26 捲動時內容蓋住視窗標題列的問題
- 修復滑鼠停在 navbar 上滾輪無法捲動頁面的問題

## v0.5.1

- 新增/編輯 Issue 時可上傳附件（支援系統檔案對話框及拖曳）
- 支援多檔案上傳，可逐一移除待上傳檔案
- 上傳過程顯示進度狀態，失敗時保留表單內容

## v0.5.0

- Issue 列表優先權欄位改為 inline 下拉選單，可直接變更優先權
- 狀態變更後自動重新載入列表，確保篩選結果正確
- 篩選按鈕文字調整：「進行中」、「已結束」
- 更新提示改為登入後才顯示
- 設定頁新增問題回報按鈕

## v0.4.1

- 修正自動更新後閃退問題
- 修正更新完成後畫面卡在 100% 無法操作
- 修正 Windows 自動更新失敗的問題

## v0.4.0

- 建立 Issue 後自動返回列表頁，不再跳到 Issue 詳情
- Issue 列表狀態欄位改為 inline 下拉選單，可直接變更狀態
- 「繼續建立」按鈕送出後自動滾動到頁面頂部
- 更新失敗時顯示實際錯誤訊息

## v0.3.1

- 修正自動更新下載失敗的問題
- 修正設定頁面更新紀錄未顯示內容的問題
- 登入頁面移除 URL 下拉，新增「記住 Redmine URL」checkbox

## v0.3.0

- 新增 System Tray：關閉視窗自動隱藏至系統匣，左鍵開啟、右鍵選單
- 新增背景 Polling：每 2 分鐘輪詢指派給自己的新 Issue
- 新增 OS 原生通知：新 Issue 時推送系統通知，點擊直接跳轉至 Issue 詳情
- 新增 Navbar 鈴鐺通知：未讀數量紅點、通知下拉列表、標記已讀、清除全部
- 通知紀錄持久化至 localStorage，登出時自動清除

## v0.2.0

- 新增自動更新功能
- 新增設定頁面：版本資訊、檢查更新按鈕、更新紀錄
- Navbar 新增齒輪圖示，有更新時顯示紅點提示
- 移除 Dashboard 底部版本號與檢查更新

## v0.1.0

- 初始版本
- 登入 Redmine（API Key 驗證）
- Dashboard：我的專案列表、我的 Issue 列表（待處理/已完成）
- 專案 Issue 列表：篩選、排序、分頁
- Issue 詳情：屬性即時編輯、歷程紀錄、附件圖片預覽
- 新增 Issue、記錄工時
- 亮色/暗色主題切換
